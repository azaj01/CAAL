# Milestones

## Completed

### v1.0 — Core Voice Assistant

**Shipped:** 2025 (pre-GSD)

**What shipped:**
- Voice conversation via LiveKit WebRTC
- Multiple LLM providers (Ollama, Groq)
- Multiple TTS providers (Kokoro, Piper)
- Multiple STT providers (Speaches, Groq Whisper)
- Home Assistant integration
- n8n MCP workflow integration
- Wake word detection (OpenWakeWord)
- Web UI with settings panel
- Mobile app (Flutter)
- Multilingual support (EN, FR, IT)

**Phases:** 1-5 (pre-GSD, not tracked)
